# Wuraola Folajimi — Systems Architecture Portfolio Platform

Production-grade multi-page portfolio ecosystem built with Next.js App Router, TypeScript, Tailwind CSS, Framer Motion, and Three.js.

## Highlights
- 40+ page architecture using static + dynamic routes.
- 6 dropdown glassmorphic navigation groups.
- 12 project detail pages and 8 case study pages.
- Blog engine with filtering/search and dynamic routes.
- Architecture docs portal, API docs, and knowledge-oriented content sections.
- Dark/light mode, scroll progress indicator, back-to-top action, sitemap, and robots metadata.

## Run
```bash
npm install
npm run dev
```

## Build checks
```bash
npm run build
```
